Консольное приложение для инкрементального расчета медианы цен из CSV-файлов с биржевыми данными.

Описание

Приложение читает CSV-файлы с историей биржевых торгов, вычисляет медиану цен по мере обработки данных и сохраняет результаты в выходной файл. Медиана рассчитывается инкрементально: каждое новое значение цены добавляется к уже обработанным, и медиана пересчитывается с учетом всех предыдущих значений.

Ключевые особенности:
- Инкрементальный расчет медианы
- Многопоточная обработка нескольких файлов
- Конфигурация через TOML-файлы
- Логирование с разными уровнями детализации

Быстрый старт

Предварительные требования
- C++23 совместимый компилятор (GCC 13+, Clang 16+, MSVC 2022+)
- CMake 3.23+
- Boost 1.82+ (Program_options, Accumulators)
- Git
